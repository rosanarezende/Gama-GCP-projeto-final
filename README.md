## Instruções para rodar o projeto

Caso possua o docker instalado na sua máquina, com ele rodando use o comando:
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install)

```bash
docker-compose up
```

<br>

<ins>OBS</ins>: É possível rodar o projeto num ambiente virtual local. O pré-requisito é ter o python3 instalado na sua máquina.

Para isso, inicialize o ambiente virtual:

```bash
# Unix ou MacOS
source ambiente_virtual/bin/activate

# No windows, entre na pasta "ambiente_virtual\Scripts" e rode
activate
```

E em seguida rode a aplicação:

```bash
python3 manage.py runserver
```
